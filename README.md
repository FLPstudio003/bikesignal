# bikesignal.sk – Web (UI základ)

Multipage web v Next.js (App Router) + TailwindCSS.

## Spustenie
```bash
npm install
npm run dev
```

## Routy
- `/` landing (CTA vstup)
- `/home` hlavná stránka
- `/o-nas`
- `/certifikaty`
- `/servis`
- `/pickup`
- `/rezervacia` (UI placeholder)
- `/kontakt`

## Poznámka
Zatiaľ je to iba web UI. Neskôr doplníme login, rezervácie (pending -> admin confirm), servisnú knižku, body, emaily.
